# Antonblast Archipelago Client

## How to Play

To play the archipelago mod you need a legal copy of ANTONBLAST in steam, after that, download the latest version of the mod here (that includes the apworld), go to the game in steam, right click and press check properties, then from there go and select the category "Game versions and betas" then select the "vm_public" version, download it, then after downloading it, go and patch your data.win of the vm_public version with the one in here (For this you need deltapatcher), after getting the data.win patched and replaced, go and download the apworld that is in this mod page with the latest version of the archipelago launcher, after this go and search "Generate Template Options" to generate the YAML of ANTONBLAST for archipelago, after running this, go to your archipelago folder inside of that folder go to "Players" then go to "Templates" and search for the yaml of Antonblast, then after that go and modify it as you like, after that go and paste it inside the players folder, after this go to the archipelago launcher and run "Generate" this will generate a multiworld with the yamls, you can do this with other yamls to create a multiworld with friends or stuff like that, after this you can go to the archipelago page and host the game in there in the "host" category or host it locally in your pc searching "Host" and opening it on the archipelago launcher, this will open the multiworld in your pc, after one of these two cases whatever is the case, just open the "antonblast client" in the launcher of archipelago and the game, after that, go search the ip (localhost if you are running host locally) or archipelago.gg if you want to join to the game in archipelago web, then after that paste the ip and the port in the client of antonblast of archipelago and then press "Connect" this should connect you to the server, after this go to the antonblast game and select the button that says "archipelago" there fill the "IP" slot, "Port" slot and the "Slot" slot (Here goes your name slot), after that you can connect if you are already connected in the client, this should start the game already, so have fun!
